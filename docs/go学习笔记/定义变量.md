### GO中定义如何定义变量

```go
// 该方法仅仅声明一个变量 并未初始化变量
var temp string
var maps map[string]string
maps["index"] // 访问 map 的 key 会报错
```

```go
// 声明一个变量并初始化
var temp1 string = "str"
var temp2 int = 1
var temp3 map[string]string = map[string]string{}
```

```go
// 短变量, 不需要显示的声明变量的类型，编译器会根据值自动推断变量所表示的类型 --常用初始化方式
temp1 := "string"
temp2 := 2
temp3 := map[string]string{}

// 多重赋值 -- 常用方式
temp4, temp5 := 4, 5
```

```go
// 该方法会初始化一个变量，并返回该变量的地址， 相当于 maps := &map[string]string{}
maps := new(map[string]string)
maps["a"] = "b" // 不会报错，
```

```go
// 该方法会初始化一个引用类型的变量 --引用类型推荐初始化的方式
maps := make(map[string]string)
slice := make([]int, 10)
channel := make(chan int)
maps["a"] = "b" // 不会报错
```
