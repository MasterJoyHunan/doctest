### 日收集数据库 LOKI

helm 添加 grafana 源

```sh
helm repo add grafana https://grafana.github.io/helm-charts
```

查看 loki 资源

```sh
> helm search repo grafana/loki
grafana/loki                        	5.41.4       	2.9.3            	(Grafana 团队推荐模式)
grafana/loki-canary                 	0.14.0       	2.9.1            	(金丝雀模式)      
grafana/loki-distributed            	0.78.0       	2.9.2            	(微服务模式)
grafana/loki-simple-scalable        	1.8.11       	2.6.1            	(读写分离模式)
grafana/loki-stack                  	2.9.11       	v2.6.1           	(单体服务模式，包含grafana,promtail)
```

使用单体模式，下载 values 配置

```sh
helm show values grafana/loki-stack > values.yaml
```

修改 values.yaml

```yaml
test_pod:
  enabled: false # 不需要

loki:
  enabled: true
  isDefault: true
  url: http://{{(include "loki.serviceName" .)}}:{{ .Values.loki.service.port }}
  readinessProbe:
    httpGet:
      path: /ready
      port: http-metrics
    initialDelaySeconds: 45
  livenessProbe:
    httpGet:
      path: /ready
      port: http-metrics
    initialDelaySeconds: 45
  datasource:
    jsonData: "{}"
    uid: ""

  # persistence: # 设置磁盘(优先使用这种方式)
  #   enabled: true
  #   storageClassName: alicloud-nas
  #   size: 20Gi
  #   accessModes:
  #     - ReadWriteOnce

  persistence: # 如果以前安装过 loki 使用创建好了的磁盘
    enabled: true
    existingClaim: loki-pvc-loki-0

  config:
    auth_enabled: false

    memberlist:
      join_members:
        # the value must be defined as string to be evaluated when secret manifest is being generating
        - '{{ include "loki.fullname" . }}-memberlist'

    ingester:
      chunk_idle_period: 3m
      chunk_block_size: 262144
      chunk_retain_period: 1m
      max_transfer_retries: 0
      wal:
        dir: /data/loki/wal
      lifecycler:
        ring:
          replication_factor: 1
    limits_config:
      enforce_metric_name: false
      reject_old_samples: true
      reject_old_samples_max_age: 168h
      max_entries_limit_per_query: 5000
    schema_config:
      configs:
        - from: 2020-10-24
          store: boltdb-shipper
          object_store: filesystem
          schema: v11
          index:
            prefix: index_
            period: 24h
    server:
      http_listen_port: 3100
      grpc_listen_port: 9095
    storage_config:
      boltdb_shipper:
        active_index_directory: /data/loki/boltdb-shipper-active
        cache_location: /data/loki/boltdb-shipper-cache
        cache_ttl: 24h # Can be increased for faster performance over longer query periods, uses more disk space
        shared_store: filesystem
      filesystem:
        directory: /data/loki/chunks
    chunk_store_config:
      max_look_back_period: 0s
    table_manager:
      retention_deletes_enabled: false
      retention_period: 0s
    compactor:
      working_directory: /data/loki/boltdb-shipper-compactor
      shared_store: filesystem
      retention_enabled: true # 新增，可以删除日志
      deletion_mode: filter-and-delete # 新增，可以删除日志
      delete_request_cancel_period: 1m # 新增，可以删除日志

promtail:
  enabled: false # 不需要
```

安装

`heml install loki grafana/loki -n istio-system -f values.yaml --version 2.9.11 ` 

升级

`heml upgrade loki grafana/loki -n istio-system -f values.yaml --version 2.9.11`
