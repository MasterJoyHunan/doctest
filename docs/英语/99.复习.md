现在

- 条件句  过去式
- would + 原型
- if I saw him, I would tell him

过去

- 条件 had 过去分词
- would have 过去分词
- if I had seen him, I would told him

将来

- 条件 should + 原型
- would + 原型
- if I should see him, I would tell him

过去
