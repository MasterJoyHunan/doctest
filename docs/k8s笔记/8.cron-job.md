### CronJob

定时任务，和 job 一样，只不过是设置某个时间点执行，可以执行多次

```yaml
apiVersion: batch/v1beta1
kind: CronJob
metadata:
  name: shttp-batch-job
spec:
  schedule: "0,15,30,45 * * * *" # 时间选择
  jobTemplate:
    spec:
      template:
        metadata:
          labels:
            beloneApp: batch
        spec:
          restartPolicy: OnFailure # 重启策略
          containers:
            - name: shttp
              image: luksa/batch-job
```
