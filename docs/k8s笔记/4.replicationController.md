### ReplicationController

用于管理 pod，当没有 pod 的时候会创建 pod， 当 pod 故障的时候会重新创建 pod。当用户指定了 pod 的数量，rc 会创建或删除 pod 到用户指定数量为止

一般也不直接使用 rc

删除 rc 不会影响 rc 创建出来的 pod

```yaml
apiVersion: v1
kind: ReplicationController
metadata:
  name: shttp-rc
spec:
  replicas: 3 # 指定副本数  
# selector: 不需要指定改选择器， rc 会直接从 template 中获取
#   beloneApp: testApp
  template:
    metadata:
      labels:
        beloneApp: testApp
        publishEnv: prod
    spec:
      containers:
        - name: shttp
          image: w4762061/shttp:v1
          ports:
            - containerPort: 8222
```
